# PV-DG Commissioning (Expo)

- **Run:** `npm install` then `npm start` / `npm run android` (device/emulator + USB debugging).
- **App:** Redux Toolkit, bottom tabs (Dashboard · Board · Site · Gateway), AsyncStorage persistence, direct LAN HTTP to ESPHome (`usesCleartextTraffic` on Android).
- **Why Expo managed:** faster delivery than bare RN; use **prebuild / dev client** when you need extra native knobs.
