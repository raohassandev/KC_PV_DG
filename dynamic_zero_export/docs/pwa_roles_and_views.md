# PWA Roles and Views

## Roles

- `user`
- `installer`
- `manufacturer`

## User

- Overview
- Energy history
- Power flow
- Connectivity
- Alerts
- Basic site settings

## Installer

- All user views
- Commissioning
- Topology
- Meter source setup
- Inverter profile setup
- Generator settings
- Policy setup
- Diagnostics
- Config review
- Export/import

## Manufacturer

- All installer views
- Profile registry
- Firmware/service
- Adapter health
- Event trace
- Support bundle
