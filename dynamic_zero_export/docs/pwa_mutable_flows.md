# PWA Mutable Flows

The Dynamic Zero Export PWA now has service-backed mutable flows for local development.

Supported flows:
- provider mode selection
- connectivity settings updates
- alert acknowledgement
- simulator state updates for live status, connectivity, alerts, and history

Role intent:
- user: mostly read-only, friendly monitoring
- installer: editable connectivity and local integration settings
- manufacturer: deeper diagnostics and support visibility
